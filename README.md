本仓库保存的是语雀文档 **《聊天系统技术文档》** 的浏览器离线页面，用于本地查阅，无需联网打开语雀。

## 内容说明

- **主文件**：`聊天系统技术文档.html`  
  双击或在浏览器中打开即可阅读（建议使用与保存时相同的相对路径，以便正确加载 `聊天系统技术文档_files/` 下的样式与脚本）。
- **资源目录**：`聊天系统技术文档_files/`  
  包含该离线页依赖的 CSS、JS 等静态资源；请勿单独移动或重命名，否则页面可能无法正常显示。

文档元信息（来自原页面）：

- 原始在线地址：<https://www.yuque.com/owqgvx/pmt0a1/bfbfog>

## 如何使用

1. 将整个项目目录保持现有结构不变。
2. 用本地浏览器打开 `聊天系统技术文档.html`。

> **说明**：这是静态快照，不是可编译运行的应用工程；若语雀原文有更新，需重新导出或保存页面以同步内容。
